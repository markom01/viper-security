# ADR 001: Brand Hierarchy — Viper Security Only

## Status
Accepted

## Context
The brand asset (insta post.jpg) shows "ELITE ADVENTURE" with an EA monogram, but the user clarified Viper Security as the main brand. The question was whether to show both, one as primary, or Viper Security only.

## Decision
Use **Viper Security only** on the website. Drop all "Elite Adventure" references from the MVP site. The content (services, pricing, fleet info) remains the same but is presented under the Viper Security brand name.

## Consequences
- Hero H1: "VIPER SECURITY" in gold serif
- No "Elite Adventure" text anywhere on the site
- The service descriptions, pricing, and routes from viper-security.md stay unchanged
- Footer: "© 2026 Viper Security"
